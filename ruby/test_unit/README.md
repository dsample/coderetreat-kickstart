Ruby: Test::Unit
================

## Prerequisites

* Ruby
* Rubygems

### Using Bundler

1. Install bundler: `gem install bundler`
2. Install bundled packages: `bundle install`

### Using Rubygems

1. Install RSpec: `gem install test-unit`

## How to run test suite

Within the `ruby/test-unit` directory, run `ruby test_gameoflife.rb`
